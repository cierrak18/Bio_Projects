------------------------------------------------------------
Project Overview
------------------------------------------------------------
This repository contains solutions to introductory bioinformatics
problems from the Rosalind platform. Rosalind is a problem-solving
resource designed to teach core concepts in computational biology,
genomics, and algorithmic thinking using real biological data.

The scripts in this repository focus on foundational DNA and population
modeling problems implemented in Python.

------------------------------------------------------------
Completed Exercises
------------------------------------------------------------

1. count_nucleotides.py
   - Problem: Counting DNA Nucleotides
   - Description:
     Counts the number of occurrences of each nucleotide
     (A, C, G, T) in a DNA string.
   - Key Concepts:
     - String processing
     - Frequency counting
     - Basic genomics concepts

2. transcribed_dna.py
   - Problem: Transcribing DNA into RNA
   - Description:
     Converts a DNA sequence into an RNA sequence by replacing
     all occurrences of thymine (T) with uracil (U).
   - Key Concepts:
     - DNA → RNA transcription
     - String manipulation
     - Biological sequence transformation

3. complement_dna.py
   - Problem: Complementing a Strand of DNA
   - Description:
     Generates the reverse complement of a DNA strand based on
     base-pairing rules (A↔T, C↔G).
   - Key Concepts:
     - Complementary base pairing
     - Reverse string operations
     - Sequence analysis

4. recurrence_rabbit.py
   - Problem: Rabbits and Recurrence Relations
   - Description:
     Models rabbit population growth using a recurrence relation,
     equivalent to a generalized Fibonacci sequence.
   - Key Concepts:
     - Recursion and iteration
     - Mathematical modeling
     - Population dynamics
